# 🚚 Last-Mile Delivery Analytics & Anomaly Detection Platform

## 🚀 Overview
This project focuses on analyzing last-mile delivery performance using SQL, Python, and Power BI. It aims to optimize logistics operations, track delivery KPIs, and detect anomalies using machine learning.

---

## 🎯 Objectives
- Monitor delivery performance using key KPIs  
- Identify delays and SLA breaches  
- Detect anomalies in delivery time and cost  
- Improve operational efficiency  

---

## 🛠️ Tech Stack
- SQL  
- Python (Pandas, Scikit-learn)  
- Power BI (DAX)  

---

## 📂 Dataset
- Logistics dataset (90K+ delivery records)  
- Includes delivery time, cost, SLA, and customer ratings  

---

## 🔍 Key Features
- KPI Tracking: On-Time Delivery %, SLA Breach Rate, Delivery Cost, CSAT  
- Power BI Dashboard for real-time monitoring  
- Anomaly Detection using Isolation Forest (~5% anomalies)  
- Delivery performance analysis  

---

## 📊 Dashboard
- Built a professional Power BI dashboard  
- Visualized delivery performance and cost trends  
- Enabled operational decision-making  

---

## 📈 Key Insights
- Delivery delays reduce customer satisfaction  
- SLA breaches increase operational costs  
- On-time delivery improves customer experience  




---

## 🚀 Future Improvements

## 💡 Conclusion
This project demonstrates how data analytics and machine learning can optimize logistics operations and improve delivery efficiency.
